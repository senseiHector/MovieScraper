Necessary Software for Deployment:
 - Python 3.6.*
 - Python virtualenv module
--------------------------------------------------------------------------------------------------
To activate virtual environment:
 - Navigate to MovieScraper/moviescraperenv/
 - enter this code into your console / terminal : source bin/activate
--------------------------------------------------------------------------------------------------
To run the Web Application on the python django web server:
 - Navigate to MovieScraper/MSProject
 - enter this code into your console / terminal : python manage.py runserver
 - If your python command line utility is linked to python 2 run : python3 manage.py runserver
 - Leave the server running and proceed to load the web application
 - Copy the url for the local django server and append the application name to it
 - enter the url in your web browser of choice. YOu should end up with the url:
 	http://127.0.0.1:8000/moviescraper/
